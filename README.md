# command_vim
vim常用命令

b：同行向前  
e：同行向后

shift+$：跳到行尾  
0：跳到行首

control+p：代码补全

split：横分屏
vsplit：竖分屏
control+w：屏幕间切换

u：还原上一步
control+r：向下一步

gg：文件头部
shift+gg：文件尾部
50+gg：定位到第50行

e+space+file：打开file，之前的文件也会打开，就是同时打开很多文件

/pattern：文件中定位pattern
定位后，shift+#: 向上翻；
shift+*：向下翻；把光标移动到其他单词上再按"shift+#"或"shift+*"将会定位光标所放的单词。

set paste：进入粘贴模式
set nopaste：退出粘贴模式
